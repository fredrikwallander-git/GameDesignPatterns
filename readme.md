# Design Patterns In Game Programming

The repo serves as course material for learning about design patterns in game programming in C++.

It also features setup information for Raylib so that it can be used to implement certain design patterns and to make a small game to make things a bit more interesting than a pure console project.

## Assignment

Your assignment for this course will be to create a clone of Vampire Survivor, though not the full game but a scoped down version that includes some key features which is defined in the Google Classroom assignment.

## Goal

At the end of this course you will have a sound grasp of design patterns, what they solve and why they exist.

You should be able to describe each pattern, identify where to use them and solve any issues that might have risen due to not using design patterns in the first place.
